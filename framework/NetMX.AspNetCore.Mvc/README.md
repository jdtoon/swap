# NetMX.AspNetCore.Mvc

This package provides features and helpers specifically for building UIs with ASP.NET Core MVC and Razor Pages, with a strong focus on enhancing the HTMX development experience.

## Key Features

-   **HTMX Response Helpers:** Provides fluent extension methods (`.WithTrigger()`, etc.) for controllers to easily construct HTMX-specific responses.
-   **Theming Abstractions:** Defines the core interfaces (`ITheme`, `IThemeManager`) for the modular and replaceable UI theming system.
